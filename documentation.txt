Section 05: Database integration & initial deployment (CI/CD setup)


33. Neon PostgreSQL database setup

-open Neon service and create database
-create new branch for development


34. Prisma 7 setup

-open link: https://nodejs.org
-update Node to latest stable version
-open link: https://www.prisma.io
-create files .env and example.env
-create variable DATABASE_URL
-type commands: npm i prisma --save-dev && npm i @prisma/client @prisma/adapter-pg pg dotenv
-open files generated by Prisma
-type command: npx prisma migrate dev --name init
-type command: npx prisma migrate dev
-type command: npx prisma db push
-open Neon db to see tables


35. Initial database migration

-type command: npm i bcryptjs && npm i @types/bcryptjs --save-dev
-in folder features create seed-spec.md file with instructions
-type command: npx prisma migrate status
-type command: npm run db:studio
-open route: http://localhost:51212


36. Seed sample data

-type command: npx prisma db seed
